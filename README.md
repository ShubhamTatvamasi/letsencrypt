# letsencrypt

Add API key of your domain provider.

issue certificate:
```
docker-compose run issue
```

get help:
```
docker-compose run help
```

Copy `fullcahin.cer` and `*.shubhamtatvamasi.com.key` to your sever.

for deleting all containers run:
```
docker-compose down
```
